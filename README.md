# OfflineImageFormatConverter
Small python ai coded app to change formats

Aplicación en Python con interfaz gráfica para convertir imágenes a distintos formatos (PNG, JPG, WEBP, BMP, ICO).

   ```bash
   git clone https://github.com/DanielaSGomez/ImageConverterPy.git
   cd ImageConverterPy
   (if needed, install recommended python)

   run:
   python conversor.py
